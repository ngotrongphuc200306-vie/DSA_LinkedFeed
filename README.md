# DSA_LinkedFeed
# Single Linked List – Social Media Post Manager

## Giới thiệu
Đây là ứng dụng WinForms sử dụng cấu trúc dữ liệu Single Linked List để mô phỏng việc quản lý các bài đăng trên mạng xã hội.  
Mỗi bài đăng được xem như một Node trong Linked List, giúp minh họa cách thêm, xóa, duyệt và tìm kiếm dữ liệu.

Ứng dụng được xây dựng phục vụ mục đích học tập môn Cấu trúc dữ liệu và giải thuật.

## Chức năng chính
- Thêm bài đăng mới
- Xóa bài đăng
- Duyệt danh sách bài đăng
- Tìm kiếm bài đăng
- Hiển thị danh sách bài đăng trên giao diện WinForms
- Lọc các bài đăng theo lượt thích
- Vẽ biểu đồ trực quan theo lượt tương tác để đưa ra các phân tích 

## Công nghệ sử dụng
- Ngôn ngữ: C#
- Nền tảng: .NET (WinForms)
- IDE: Visual Studio
- Cấu trúc dữ liệu: Single Linked List

## Hướng dẫn cài đặt và chạy chương trình
1. Vào repository GitHub
2. Chọn Code → Download ZIP
3. Chuột phải vào file ZIP → Properties → tích Unblock → OK
4. Giải nén toàn bộ file
5. Mở Visual Studio
6. Chọn File → Open → Project or Solution
7. Mở file SingleLinkList.sln
8. Nhấn Run để chạy chương trình

## Lưu ý
- Cần có Visual Studio để chạy chương trình
- Nếu gặp lỗi resource (.resx), hãy kiểm tra việc Unblock file ZIP trước khi giải nén
